# Hướng Dẫn Sử Dụng Hệ Thống Interactive Elements

## 📋 Tổng Quan

File `css/interactive-elements.css` cung cấp hệ thống buttons, forms, và interactions chuẩn hóa cho toàn bộ dự án Spa ANA.

---

## 🎯 Hệ Thống Buttons

### 1. Semantic Color Usage

| Loại Action | Màu Sắc | Class | Icon | Dùng cho |
|------------|---------|-------|------|----------|
| Primary Action | Xanh dương | `btn-primary` / `btn-outline-primary` | `fa-eye`, `fa-edit`, `fa-save` | Xem chi tiết, Chỉnh sửa, Lưu |
| Success Action | Xanh lá | `btn-success` / `btn-outline-success` | `fa-check`, `fa-check-circle` | Xác nhận, Hoàn thành |
| Danger Action | Đỏ | `btn-danger` / `btn-outline-danger` | `fa-trash`, `fa-times`, `fa-times-circle` | Hủy, Xóa |
| Secondary Action | Xám | `btn-secondary` / `btn-outline-secondary` | `fa-arrow-left`, `fa-times` | Quay lại, Đóng |
| Warning Action | Vàng cam | `btn-warning` / `btn-outline-warning` | `fa-redo`, `fa-exclamation-triangle` | Đặt lại, Lưu thay đổi |
| Info Action | Xanh nhạt | `btn-info` / `btn-outline-info` | `fa-info-circle`, `fa-question-circle` | Thông tin, Xem lý do |

### 2. Button Sizes

```html
<!-- Small -->
<button class="btn btn-primary btn-sm">Nút nhỏ</button>

<!-- Medium (default) -->
<button class="btn btn-primary">Nút vừa</button>

<!-- Large -->
<button class="btn btn-primary btn-lg">Nút lớn</button>
```

### 3. Icon-Only Buttons

```html
<!-- Icon only - standard size -->
<button class="btn btn-primary btn-icon-only" title="Xem chi tiết" aria-label="Xem chi tiết">
    <i class="fas fa-eye"></i>
</button>

<!-- Icon only - small -->
<button class="btn btn-primary btn-sm btn-icon-only" title="Xem chi tiết" aria-label="Xem chi tiết">
    <i class="fas fa-eye"></i>
</button>
```

**QUAN TRỌNG:**
- Icon-only buttons **BẮT BUỘC** có `title` attribute
- Icon-only buttons **BẮT BUỘC** có `aria-label` attribute

## 📊 Table Action Buttons

### Standard Structure

```html
<td class="table-actions">
    <button class="table-action-btn btn-primary" 
            title="Xem chi tiết" 
            aria-label="Xem chi tiết lịch hẹn"
            onclick="viewDetails('LH001')">
        <i class="fas fa-eye"></i>
    </button>
    
    <button class="table-action-btn btn-success" 
            title="Xác nhận" 
            aria-label="Xác nhận lịch hẹn"
            onclick="confirmAppointment('LH001')">
        <i class="fas fa-check"></i>
    </button>
    
    <button class="table-action-btn btn-danger" 
            title="Hủy" 
            aria-label="Hủy lịch hẹn"
            onclick="cancelAppointment('LH001')">
        <i class="fas fa-times"></i>
    </button>
</td>
```

### Action Mapping by Status

| Status | Actions | Button Classes |
|--------|---------|---------------|
| Chờ xác nhận | Xác nhận, Hủy | `btn-success`, `btn-danger` |
| Đã xác nhận | Chỉnh sửa, Hủy | `btn-primary`, `btn-danger` |
| Đang xử lý | Hủy | `btn-danger` |
| Hoàn thành | Xem chi tiết, In | `btn-info`, `btn-secondary` |
| Đã hủy | Đặt lại, Xem lý do | `btn-warning`, `btn-info` |

## 🎨 Form Elements

### Required Fields

```html
<div class="mb-3">
    <label class="form-label required" for="name">Họ tên</label>
    <input type="text" class="form-control" id="name" required>
</div>
```

Kết quả: Label sẽ hiển thị "Họ tên *" với màu đỏ

### Validation Messages

```html
<div class="mb-3">
    <label class="form-label required" for="phone">Số điện thoại</label>
    <input type="tel" class="form-control" id="phone" required>
    <div class="invalid-feedback" id="phoneError">
        Vui lòng nhập số điện thoại hợp lệ (10 chữ số)
    </div>
</div>
```

Show validation error:
```javascript
document.getElementById('phone').classList.add('is-invalid');
document.getElementById('phoneError').classList.add('show');
```

Clear validation error:
```javascript
document.getElementById('phone').classList.remove('is-invalid');
document.getElementById('phoneError').classList.remove('show');
```

## ⚙️ Loading States

### Button Loading

```html
<button class="btn btn-primary" id="submitBtn" onclick="handleSubmit()">
    <i class="fas fa-save me-2"></i>Lưu thay đổi
</button>
```

```javascript
function handleSubmit() {
    const btn = document.getElementById('submitBtn');
    
    // Add loading state
    btn.classList.add('loading');
    btn.disabled = true;
    
    // Simulate API call
    setTimeout(() => {
        // Remove loading state
        btn.classList.remove('loading');
        btn.disabled = false;
        
        // Show success message
        alert('Lưu thành công!');
    }, 1000);
}
```

### Spinner Only

```html
<div class="loading-spinner"></div>
```

## 📦 Modal Confirmations

### Destructive Action Confirmation

```html
<!-- Modal HTML -->
<div class="modal fade" id="deleteModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">
                    <i class="fas fa-exclamation-triangle text-danger me-2"></i>
                    Xác nhận xóa
                </h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">
                <p>Bạn có chắc chắn muốn xóa mục này không?</p>
                <p class="text-muted small">
                    <i class="fas fa-info-circle me-1"></i>
                    Hành động này không thể hoàn tác.
                </p>
                <div class="mb-3">
                    <label class="form-label required">Lý do xóa</label>
                    <textarea class="form-control" id="deleteReason" rows="2"></textarea>
                </div>
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">
                    <i class="fas fa-times me-2"></i>Hủy
                </button>
                <button type="button" class="btn btn-danger" onclick="confirmDelete()">
                    <i class="fas fa-trash me-2"></i>Xóa
                </button>
            </div>
        </div>
    </div>
</div>
```

```javascript
function showDeleteModal(id) {
    document.getElementById('deleteReason').value = '';
    document.getElementById('deleteItemId').value = id;
    new bootstrap.Modal(document.getElementById('deleteModal')).show();
}

function confirmDelete() {
    const reason = document.getElementById('deleteReason').value.trim();
    
    if (!reason) {
        alert('Vui lòng nhập lý do xóa!');
        return;
    }
    
    // Perform delete action...
    console.log('Deleting with reason:', reason);
    
    // Close modal
    bootstrap.Modal.getInstance(document.getElementById('deleteModal')).hide();
    
    // Show success toast
    alert('Đã xóa thành công!');
}
```

## 🎯 Best Practices

### 1. Always Use Semantic Colors

✅ **CORRECT:**
```html
<button class="btn btn-danger">Xóa</button>
<button class="btn btn-success">Xác nhận</button>
```

❌ **WRONG:**
```html
<button class="btn btn-primary" style="background-color: red;">Xóa</button>
```

### 2. Icon-Only Buttons MUST Have Tooltips

✅ **CORRECT:**
```html
<button class="btn btn-primary btn-icon-only" 
        title="Xem chi tiết" 
        aria-label="Xem chi tiết">
    <i class="fas fa-eye"></i>
</button>
```

❌ **WRONG:**
```html
<button class="btn btn-primary btn-icon-only">
    <i class="fas fa-eye"></i>
</button>
```

### 3. Destructive Actions MUST Confirm

✅ **CORRECT:**
```html
<button class="btn btn-danger" onclick="showDeleteConfirm()">
    <i class="fas fa-trash me-2"></i>Xóa
</button>
```

❌ **WRONG:**
```html
<button class="btn btn-danger" onclick="deleteItem()">
    <i class="fas fa-trash me-2"></i>Xóa
</button>
```

### 4. Form Validation MUST Be Clear

✅ **CORRECT:**
```html
<div class="invalid-feedback show">
    Vui lòng nhập số điện thoại hợp lệ
</div>
```

❌ **WRONG:**
```javascript
alert('Sai!');
```

### 5. Loading States for Async Actions

✅ **CORRECT:**
```javascript
btn.classList.add('loading');
btn.disabled = true;
```

❌ **WRONG:**
```javascript
// No loading state, user doesn't know what's happening
```

## 📋 Checklist Khi Code

Trước khi commit code, kiểm tra:

- [ ] Buttons sử dụng semantic colors đúng
- [ ] Icon-only buttons có `title` và `aria-label`
- [ ] Destructive actions có confirmation modal
- [ ] Forms có validation messages rõ ràng
- [ ] Async actions có loading states
- [ ] Toast notifications hiển thị sau actions
- [ ] Focus states visible (keyboard navigation)
- [ ] Responsive design tested (mobile, tablet)
- [ ] Accessibility checked (screen reader friendly)

## 🔧 Troubleshooting

### Buttons không align trong table

**Sử dụng class `.table-actions`:**
```html
<td class="table-actions">
    <!-- buttons here -->
</td>
```

### Tooltips không hiển thị

**Khởi tạo Bootstrap tooltips:**
```javascript
document.addEventListener('DOMContentLoaded', function() {
    const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
    tooltipTriggerList.forEach(function(tooltipTriggerEl) {
        new bootstrap.Tooltip(tooltipTriggerEl);
    });
});
```

### Loading spinner không quay

**Kiểm tra animation:**
```css
@keyframes spin {
    to { transform: rotate(360deg); }
}
```
